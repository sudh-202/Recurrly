# Recurrly - Subscription Management Made Simple 💸

Recurrly is a premium subscription management mobile application built with **React Native**, **Expo**, and **NativeWind**. It helps users take control of their recurring expenses, track billing cycles, and gain insights into their spending habits.

## ✨ Features

- **Subscription Tracking**: Easily add and manage all your recurring services (Netflix, Spotify, Gym, etc.) in one place.
- **Billing Alerts**: Stay ahead of your bills with timely notifications before a subscription renews.
- **Spending Insights**: Visualize your monthly and yearly subscription costs with intuitive charts and breakdowns.
- **Smart Categorization**: Group your subscriptions by category for better organization.
- **Beautiful UI**: Modern, sleek interface with support for dark mode and premium aesthetics.
- **Cross-Platform**: Seamless experience on both iOS and Android.

## 🚀 Tech Stack

- **Framework**: [Expo](https://expo.dev) / [React Native](https://reactnative.dev)
- **Routing**: [Expo Router](https://docs.expo.dev/router/introduction) (File-based routing)
- **Styling**: [NativeWind](https://www.nativewind.dev) (Tailwind CSS for React Native)
- **Icons**: [Expo Vector Icons](https://docs.expo.dev/guides/icons/)
- **Language**: [TypeScript](https://www.typescriptlang.org)

## 🛠️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sudh-202/Recurrly.git
   cd Recurrly
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Run on your device**
   - Download the **Expo Go** app on your iOS or Android device.
   - Scan the QR code appearing in your terminal.

## 📂 Project Structure

- `app/`: Contains the main application routes and screens.
  - `(tabs)/`: Main navigation and dashboard screens.
  - `(auth)/`: Authentication flow (Sign In / Sign Up).
- `constants/`: Global style constants and configuration.
- `hooks/`: Custom React hooks for global functionality.
- `assets/`: Images, fonts, and other static assets.
